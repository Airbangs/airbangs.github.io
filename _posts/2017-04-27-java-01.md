一、常用控制输出符

\t 输出8个空格，相当于一个制表位

\n 换行符

例：输出日历
'''
public class ltcLearning002{
	public static void main(String[] args){
		System.out.print("日\t一\t二\t三\t四\t五\t六\n");
		System.out.print("\t\t\t1\t2\t3\t4\n");
		System.out.print("26\t27\t28\t29\t30\t31\t\t\n");
	}
}
